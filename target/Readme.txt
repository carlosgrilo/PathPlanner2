Passos:
1. Correr com o comando: java -jar AlgoritmoGenetico-1.0-SNAPSHOT.jar
2. Para que o programa fique em modo de comunicação carregar na checkbox "Communication" e esperar até ficar selecionada.(Pode demorar uns segundos)
3. Gerar um problema com o botão RNG
4. Correr o problema com o botão GA
5. No fim do calculo o programa vai enviar automaticamente uma mensagem.

Para ver se não existem erros verificar na consola onde a aplicação foi iniciada.
